For task1 and task 2
There are 3 variable above main function
	1. Path: path to the folder which contain train and test folder
	2. PATH_TO_LOAD_MODEL: the folder path from where saved network can be loaded.
	3. PATH_TO_SAVE_MODEL: path where the trained model should be saved.
	4. PATH_TO_SAVE_IMAGE: path where images after each iteration should be saved.
After setting these variable run the file.
